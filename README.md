# openAll

Open multiple URLs in new browser tabs with one click. A simple start page for intranet or internet use.

**Live:** [openall.jphe.in](https://openall.jphe.in)

## How It Works

1. Add URLs to the list
2. Click **GO!**
3. All URLs open in new tabs

Built with jQuery. Make sure to allow pop-ups from the page in your browser.
